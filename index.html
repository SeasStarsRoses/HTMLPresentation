What is HTML?

  1. Describes the structure and contents of a document
  2. Offers tags like headings, lists, tables, ...
  3. The document can be rendered in a browser for viewing

Advantages

  1. Easy to learn
  2. Can be written with a simple text editor
  3. Running with all platforms, operating systems, browsers

Disadvantages

  1. The view depends on the browser
  2. It is not always possible to predict the layout
  3. It is hard for web designers to use a fixed layout
